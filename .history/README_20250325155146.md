# MyLocalApp - Village Community Platform

A React Native mobile application designed to strengthen community spirit in rural villages by providing hyper-localized information and interactivity.

## Features

### Home Screen
- Search functionality with consistent design
- Filter buttons for content categories
- Local news articles feed
- Pre-populated content from local sources

### Chat Screen
- Open chat groups for village communication
- Organization broadcast channels
- AI-powered village information chatbot
- Real-time messaging interface

### Calendar Screen
- Interactive event calendar with date strip
- Event filtering by categories
- Detailed event cards with time, location, and category
- Pre-populated events from local sources

### Map Screen
- Interactive village map
- Points of interest markers
- Location search functionality
- Category-based filtering

### Profile Screen
- User profile management
- Organization account upgrade option
- Account settings and preferences
- Privacy and security settings

## Project Structure

## Dependencies

- React Native 0.72.6
- React Navigation (Bottom Tabs)
- React Native Maps
- React Native Vector Icons
- React Native Calendar Strip
- React Native Safe Area Context
- React Native Screens

## Setup Instructions

1. Install dependencies:
\`\`\`bash
npm install
\`\`\`

2. For iOS, install pods:
\`\`\`bash
cd ios && pod install && cd ..
\`\`\`

3. Start the application:
\`\`\`bash
# iOS
npm run ios

# Android
npm run android
\`\`\`
